# 2차 프로젝트
> 팀 프로젝트

- 프로젝트명 : 명화 분석기 (인원 3명)
- 기간 : 2025.02.17 ~ 2025.03.07 (3주)
- 언어 : Python 3.12.2
- IDE : Anaconda Jupyter notebook7.2.2, pycharm 3.1.1
- Library : tensorflow, numpy, pandas, scikit-learn, pillow, dJango 등
- 프로젝트 소개 : 명화 이미지 데이터를 활용한 데이터 분석, 딥러닝과 머신러닝을 통해 화가, 스타일, 장르를 예측하는 시스템
